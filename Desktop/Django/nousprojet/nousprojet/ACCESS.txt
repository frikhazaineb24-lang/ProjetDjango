🧪 TESTER L'APPLICATION

1️⃣  INSCRIPTION DONNEUR
   - Accéder: http://127.0.0.1:8000/inscription/donneur/
   - Créer un compte donneur
   - Groupe sanguin: O+, A+, etc.

2️⃣  INSCRIPTION HÔPITAL
   - Accéder: http://127.0.0.1:8000/inscription/hopital/
   - Créer un compte hôpital
   - Le compte sera en attente de validation admin

3️⃣  VALIDATION ADMIN
   - Accéder: http://127.0.0.1:8000/admin/
   - Se connecter avec: admin / admin@2026
   - Aller à Hopitals
   - Cocher "Valide" pour un hôpital
   - L'hôpital reçoit une notification

4️⃣  UTILISER LES FONCTIONNALITÉS
   - Donneur: Voir demandes, s'inscrire, enregistrer don
   - Hôpital: Créer demande, organiser campagne
   - Admin: Gérer tous les modèles

🔧 COMMANDES UTILES

Arrêter le serveur:
  • Appuyer sur CTRL + C (ou CTRL + Break)

Redémarrer le serveur:
  • Appuyer sur CTRL + C
  • Relancer: python manage.py runserver

Créer un nouvel administrateur:
  • python manage.py createsuperuser

Voir les logs:
  • Le serveur affiche les logs en temps réel

Réinitialiser la base:
  • Supprimer: db.sqlite3
  • Relancer: python manage.py migrate
