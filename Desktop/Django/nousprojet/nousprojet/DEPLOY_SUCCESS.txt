🧪 SCÉNARIO DE TEST COMPLET:

1. Admin Dashboard
   - Accéder: http://127.0.0.1:8000/admin/
   - Login: admin / admin@2026
   - Voir: Tous les modèles disponibles

2. Créer un Donneur
   - Accéder: http://127.0.0.1:8000/inscription/donneur/
   - Remplir: Prénom, Nom, Email, Groupe sanguin (O+), Ville
   - Confirmer: Accès à Dashboard Donneur

3. Créer un Hôpital
   - Accéder: http://127.0.0.1:8000/inscription/hopital/
   - Remplir: Nom, Adresse, Ville, Agrément
   - Résultat: En attente de validation

4. Valider l'Hôpital
   - Admin Dashboard
   - Hopitals → Cocher "Valide" → Save
   - Résultat: Hôpital peut se connecter

5. Créer une Demande Urgente
   - Login Hôpital
   - "Ajouter une demande" → Groupe: O+, Quantité, Délai
   - Résultat: Appel visible sur Dashboard Donneur

6. Donneur Répond à l'Appel
   - Login Donneur
   - "Répondre" à la demande
   - Résultat: Réponse enregistrée

7. Créer une Campagne
   - Login Hôpital
   - "Créer Campagne" → Nom, Date, Créneau
   - Résultat: Campagne visible pour donneurs

8. S'inscrire à Campagne
   - Login Donneur
   - Campagnes → S'inscrire → Choisir créneau
   - Résultat: Inscription confirmée
